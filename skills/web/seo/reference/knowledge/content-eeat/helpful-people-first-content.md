# The helpful-content system is now core ranking — the Who/How/Why self-assessment

The former standalone "Helpful Content Update" was absorbed into Google's continuous core ranking system in March 2024; content is assessed as "people-first" via the Who/How/Why questions, not by a switch that runs periodically.

## Facts
- The helpful-content system is **no longer a separate signal or periodic update** — as of the March 2024 core update it is part of the always-on core ranking system. There is no longer a discrete "HCU" to recover from; recovery tracks general core updates. [Google core-update-2024-03]
- Google's self-assessment is **Who / How / Why**: (Who) is authorship clear, with a byline where readers would expect one that links to real author background; (How) is any automation/AI use disclosed and explained; (Why) is the content made primarily to help people, not primarily to rank. "Primarily to attract search visits" is the failure mode. [Google helpful-content]
- "People-first" ≠ "search-engine-first". Content built for people that also happens to be findable is the target; content reverse-engineered from what ranks is what the system demotes. [Google helpful-content]
- A byline "where one would be expected" is Google's stated expectation, and it should lead to real author info — but the byline itself is an E-E-A-T *signal to raters*, not a mechanical ranking bonus. See freshness-and-authorship. [Google helpful-content]
